# Robotics School Database - Activity 7

## Project Name
Robotics School Platform Database

## Project Description
This project models a database for a robotics school platform. The system allows administrators, teachers and students to interact with robotics courses and educational materials.

The database includes entities such as users, roles, groups, courses and robotics kits.

## Seed Data
The database includes seed data with:

- 3 users (Administrative, Teacher and Student)
- 3 robotics kits
- 100 generated robotics courses
